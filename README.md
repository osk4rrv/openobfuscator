# OpenObfuscator.us

**OpenObfuscator** is an open-source LuaJIT obfuscator designed to make your code harder to analyze, reverse engineer, or modify. It transforms readable LuaJIT scripts into an obfuscated non-readable file keeping all functions from input code.
